ChatIdentityAdapter (lib/chat-ui/adapters/types.ts, to be created in Plan 02):
- interface ChatIdentityAdapter {
-   endpoint: string;
-   getAuthHeaders(): Promise<Record<string, string>>;
-   getRequestBody(): Promise<Record<string, unknown>>;
- }

EmbeddedAdapter (lib/chat-ui/adapters/embedded.ts, to be created in Plan 02):
- class EmbeddedAdapter implements ChatIdentityAdapter
- readonly endpoint = '/api/chat'
- getAuthHeaders(): calls `await shopify.idToken()` (runtime global) and returns { Authorization: `Bearer ${token}` }
- getRequestBody(): returns {}

StorefrontAdapter (lib/chat-ui/adapters/storefront.ts, to be created in Plan 02):
- class StorefrontAdapter implements ChatIdentityAdapter
- readonly endpoint = '/api/proxy/chat'
- STORAGE_KEY = 'smartdiscovery.visitor_id'
- getAuthHeaders(): returns {}
- getRequestBody(): if window undefined → {}; else read localStorage[STORAGE_KEY], if absent generate via crypto.randomUUID() + persist, return { visitor_id: '<uuid>' }

LocalStorageHistoryStore (lib/chat-ui/stores/local-storage.ts, to be created in Plan 02):
- constructor(scope: string) — throws Error('LocalStorageHistoryStore requires a non-empty scope') if !scope (T-5-01)
- key = `smartdiscovery.history.${scope}` (HISTORY_CAP = 10)
- list/add/clear/subscribe per HistoryStore interface

LocalStorageSavedProductsStore — same shape, key `smartdiscovery.saved.${scope}`, no cap, toggle/has instead of add.
