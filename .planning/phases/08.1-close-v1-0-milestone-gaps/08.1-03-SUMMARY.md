## What Was Built

### StorefrontDrawer.tsx — Final Shape

**Outer component `StorefrontDrawer`:** Public signature unchanged (`StorefrontDrawerProps` with `shop?`, `visitorId?`, `customerId?`, `accent?`, `position?`, `initialOpen?`). FAB markup, aria-label toggling, drawer aside, header, close button, tablist, and Escape-key handler are byte-equivalent to the pre-plan version. The `role="tabpanel"` body now conditionally renders:

- When `shop && visitorId` are both truthy: mounts `<DrawerBody activeTab={activeTab} shop={shop} visitorId={visitorId} customerId={customerId ?? null} />`
- Otherwise: renders the original placeholder `<p>` elements ("Chat coming up…", "No history yet.", "No saved products yet.")

**Inner component `DrawerBody`:** Non-exported function component owning all hook calls. Instantiates `new StorefrontAdapter()` via `React.useMemo(..., [])`. Calls `useDbBackedHistoryStore({ shop, visitorId, customerId })` and `useDbBackedSavedProductsStore({ shop, visitorId, customerId })`. Derives `savedProductIds` via `React.useMemo`. Switches on `activeTab` to return `ChatPane`, `HistoryPanel`, or `SavedProductsPanel` with props mirroring `app/(embedded)/chat/chat-shell.tsx`.

**Imports added:**
```tsx
import { ChatPane, HistoryPanel, SavedProductsPanel, useDbBackedHistoryStore, useDbBackedSavedProductsStore } from '@/lib/chat-ui';
import { StorefrontAdapter } from '@/lib/chat-ui/adapters/storefront';
```
