POST capture (modify the existing handleStartSync):

Polling effect (new):
```typescript
useEffect(() => {
  if (!syncRunId || (syncState && ['succeeded', 'partial', 'failed'].includes(syncState))) return;
  const id = setInterval(async () => {
    const token = await shopify.idToken();
    const res = await fetch(`/api/shopify/sync/status?syncRunId=${syncRunId}`, {
      headers: { Authorization: `Bearer ${token}` },
    });
    if (res.ok) {
      const data = await res.json();
      setSyncState(data.state);
      setProcessedCount(data.processedCount);
      setTotalCount(data.totalCount ?? null);
      setErrors(data.errors ?? []);
    }
  }, 2000);
  return () => clearInterval(id);
}, [syncRunId, syncState]);
```

Render structure (replace the existing start-sync button conditional):
- If `syncRunId === null`: show the existing `<s-button data-testid="start-sync">Start sync</s-button>` (unchanged).
- Else: show progress view.
  - `<s-progress-bar value={totalCount ? Math.round((processedCount / totalCount) * 100) : 0}>` (hide or use indeterminate when totalCount null — `value={0}` is the simplest V1 choice).
  - `<s-text>{totalCount ? `${processedCount} / ${totalCount} products` : `${processedCount} products synced so far`}</s-text>`
  - State badge: `<s-badge>{stateLabel(syncState)}</s-badge>` where stateLabel maps queued→'Queued', running→'Running', succeeded→'Succeeded', partial→'Partial', failed→'Failed'.
  - If `syncState === 'succeeded'`: `<s-banner tone="success">Your store is ready — {processedCount} products synced</s-banner>` + `<s-button variant="primary" href="/chat">Open admin chat</s-button>`.
  - If `syncState === 'partial'`: `<s-banner tone="warning">{processedCount} products synced, {errors.length} failed</s-banner>` + `<s-button onClick={handleStartSync}>Retry sync</s-button>`.
  - If `syncState === 'failed'`: `<s-banner tone="critical">Sync failed</s-banner>` + `<s-button onClick={handleStartSync}>Retry sync</s-button>`.

D-13 locks 2000ms polling interval (constant; no exponential backoff in V1).
